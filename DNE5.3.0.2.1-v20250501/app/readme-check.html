CHECK : How to use samples

1. Please check the product model of your robot.
Communication is done by sending and receiving MQTT messages using JavaScript.
MQTT messages and some system features are only available for E-series or other system integration products.
In some simpler product line-ups, users can NOT use these applications, and the MQTT interface is NOT supported.

2. Please enable Cookie on your browser.
If enable Cookie, for example "address" is displayed fine on top page.

3. Check the string at the top of the each sample application.
This indicates the connection status.
If MQTT on Websocket is connected successfully, the unit serial number will be displayed.

4. For other trouble shooting.
Please check with the distributor or the contact listed in the instruction manual.
